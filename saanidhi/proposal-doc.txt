 1. Project Overview

**SachetAI** is a disaster-response AI assistant designed for **NDRF operators and emergency responders**.
The frontend simulates a **real-time operational dashboard** that integrates:

* Conversational AI (text + voice)
* Live disaster alerts
* Interactive Maharashtra map with hotspots
* Emergency modules (Floods, Cyclones, Evacuation, Shelters)
* Backend-ready RAG + FastAPI integration

This frontend is built to be **fully functional visually** while remaining **backend-agnostic**, enabling smooth integration later.

* **Maps:** Leaflet + OpenStreetMap

 7. API Integration Design 

The frontend is intentionally designed to keep **all backend calls isolated** inside `/src/api`.

 7.1 `voiceRecorder.js`

**Purpose**

* Handle microphone permission
* Start/stop audio recording
* Return `audioBlob`

 7.2 `chatApi.js` (Backend Integration Point)

**Purpose**

* Central communication layer
* Backend  only modifies URLs & payloads
* UI remains untouched

#### Example Structure

```js
// src/api/chatApi.js

export async function sendChatMessage(message) {
  // Backend team will replace URL
  const response = await fetch("http://BACKEND_URL/chat", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ query: message }),
  });

  return response.json();
}

export async function sendVoiceMessage(audioBlob) {
  const formData = new FormData();
  formData.append("audio", audioBlob);

  const response = await fetch("http://BACKEND_URL/voice", {
    method: "POST",
    body: formData,
  });

  return response.json();
}
```

âš ï¸ URLs are placeholders
âš ï¸ No API keys used
âš ï¸ Fully FastAPI compatible
